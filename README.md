## 6.代码

- 行内代码用反引号 \`代码\`，实际上\`\`里的内容不会转化markdown

e.g `cin>>a>>endl;`

- 还有一种是代码块，需要三个反引号

例子如下：

\```

#include<stdio.h>

int main(){

...

}

\```


## 7.表格

``````markdown

``````

虽然不是很对齐，但是markdown会自动对齐
